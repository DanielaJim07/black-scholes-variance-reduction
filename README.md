# Black-Scholes vs Monte Carlo Option Pricing

This project evaluates option pricing methods using:

- Black-Scholes analytical formula
- Monte Carlo simulation
- Variance reduction techniques (antithetic variables, control variates)

## Objective
Compare pricing accuracy, variance, and computational efficiency.

## Methods
- Historical data from yfinance
- Maximum Likelihood Estimation (MLE)
- Geometric Brownian Motion modeling
- Monte Carlo simulation
- Variance reduction techniques

## Technologies
Python, NumPy, Pandas, Matplotlib, SciPy

## How to run
pip install -r requirements.txt  
python main.py

## Key Insight
Variance reduction methods significantly improve Monte Carlo efficiency by reducing estimator variance and improving accuracy.
